# js-plugin-example

Web 前端插件文档库示例代码，文档库地址：https://www.yuque.com/pengloo53/web

## 介绍

如今，随着大前端的发展，曾经的霸主：jQuery 也渐渐的陨落，随之一起没落的，也包含基于 jQuery 的一些优秀的插件。这些优秀的插件，让我们在前端的开发中，节省了大量的时间。自己的项目中也使用了大部分基于 jQuery 的前端插件，帮助我实现了很多的功能，节省了大量的开发时间。

所以，我计划发起一个开源项目，整理并收集这些优秀的前端插件的使用方法，并提供示例代码，可供调试和学习。一来为了留个存档，二来也是为了分享出去，帮助到需要的人。当然，我希望你也能跟我一起维护这个文档，让它帮助到更多的人。

> 该文档起于 js 框架，但是不止于 js，希望能整理所有的 Web 前端框架。

- 如果有兴趣可以一起维护文档，文档协作邀请链接：https://www.yuque.com/g/pengloo53/web/collaborator/join?token=rMudjrT0PjYZv8BR
- 为了文档的可用性，我希望每一个插件都有一篇独立的介绍文档，以及独立的页面演示。该 repo 就是在线演示的源代码，repo 地址：[pengloo53/js-plugin-example](https://github.com/pengloo53/js-plugin-example)
- 在线演示地址：http://js.90byte.com

## 部署
1. 安装 Node 环境
2. 安装 supervisor，安装方法：`npm i -g supervisor`
3. 安装 bower 包管理器，安装方法：`npm i -g bower`
4. 克隆代码到本地，顺便点个 Star
5. 进入项目目录，安装依赖，`npm install` 以及 `bower install`
6. 运行项目，`npm start`
7. 浏览器打开项目：http://localhost:3030
